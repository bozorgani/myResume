# راهنمای اتصال Google Search Console به سایت

## مرحله 1: دریافت کد تأیید از Google Search Console

1. به [Google Search Console](https://search.google.com/search-console) بروید
2. با حساب Google خود وارد شوید
3. روی دکمه **"Add Property"** کلیک کنید
4. گزینه **"URL prefix"** را انتخاب کنید
5. آدرس کامل سایت خود را وارد کنید (مثلاً: `https://bozorgani.ir`)
6. روی **"Continue"** کلیک کنید
7. در صفحه تأیید، روش **"HTML tag"** را انتخاب کنید
8. کد داخل `content` را کپی کنید (مثلاً: `abc123xyz789...`)

## مرحله 2: افزودن کد به پروژه

1. در پوشه `myResume` یک فایل به نام `.env.local` ایجاد کنید
2. محتوای زیر را در آن قرار دهید:

```env
NEXT_PUBLIC_GOOGLE_SEARCH_CONSOLE=کد-تأیید-که-کپی-کردید
```

**مثال:**
```env
NEXT_PUBLIC_GOOGLE_SEARCH_CONSOLE=abc123xyz789def456ghi012
```

⚠️ **توجه:** فایل `.env.local` را به `.gitignore` اضافه کنید تا کد تأیید شما در Git ذخیره نشود.

## مرحله 3: راه‌اندازی مجدد سرور

1. اگر سرور در حال اجرا است، آن را متوقف کنید (Ctrl+C)
2. دوباره سرور را اجرا کنید:

```bash
npm run dev
```

## مرحله 4: تأیید در Google Search Console

1. به صفحه تأیید در Google Search Console برگردید
2. روی دکمه **"Verify"** کلیک کنید
3. اگر همه چیز درست باشد، پیام موفقیت را خواهید دید

## مرحله 5: بررسی

برای اطمینان از اینکه کد به درستی اضافه شده:

1. سایت خود را در مرورگر باز کنید
2. روی صفحه راست کلیک کنید و **"View Page Source"** را انتخاب کنید
3. در بخش `<head>` دنبال تگ زیر بگردید:

```html
<meta name="google-site-verification" content="کد-تأیید-شما" />
```

## نکات مهم

- ✅ کد تأیید باید دقیقاً همان چیزی باشد که از Google Search Console کپی کرده‌اید
- ✅ فایل `.env.local` باید در پوشه `myResume` باشد (هم‌سطح `package.json`)
- ✅ بعد از تغییر `.env.local` حتماً سرور را راه‌اندازی مجدد کنید
- ✅ در محیط Production (هنگام Deploy) باید این متغیر محیطی را در پنل هاستینگ خود نیز اضافه کنید

## عیب‌یابی

اگر تأیید انجام نشد:

1. مطمئن شوید فایل `.env.local` درست ایجاد شده
2. مطمئن شوید کد تأیید را بدون فاصله اضافی کپی کرده‌اید
3. سرور را راه‌اندازی مجدد کنید
4. Cache مرورگر را پاک کنید
5. در View Source صفحه، بررسی کنید که تگ meta اضافه شده باشد
